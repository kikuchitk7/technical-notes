Amazon SageMaker Autopilot
==================================

Amazon SageMaker Autopilot とは
------------------------------------
- 機械学習モデルを自動で構築・学習できる。
- 表形式のデータに対する予測と分類の機械学習の AutoML 機能。
- 多数のユースケースに適用可能。

    - 不正検知
    - ユーザ分類 (離反予測)
    - レコメンデーション
    - etc.

- データの前処理、アルゴリズムの選択、インスタンスタイプの選択なども自動化。
- 東京リージョンで利用可能。

AutoML とは
--------------------
- Automated Machine Learning の略。
- 機械学習モデルの設計・構築を自動化するための手法全般、またはその概念を指す。
- 特に、Google が注力しており、2017年5月に `Using Machine Learning to Explore Neural Network Architecture <https://ai.googleblog.com/2017/05/using-machine-learning-to-explore.html>`_ という記事を出している。
- Amazon Forecast も複数アルゴリズムの中から自動選択されるが、これを AutoML としている。

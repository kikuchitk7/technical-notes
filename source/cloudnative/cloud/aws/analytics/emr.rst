Amazon EMR
=====================

メモ
--------------
- EMRFS 独自ストレージ 実体は S3
- インスタンスグループ

    - Master
    
        - クラスタの可用性に関わるため、Spot instance の利用に適さない。
    
    - Core
    
        - HDFS を持っている
        - スケールイン時にデコミッションが必要。
        - Spot instance の利用に適さない。
        
    - Task

要調査
-----------------
- HDFS
- インスタンスフリート
- スポットブロック

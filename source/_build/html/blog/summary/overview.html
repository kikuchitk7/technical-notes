[第2回] AWS の機械学習サービスとは¶

はじめに¶

前回 は、本連載の背景についてご説明し、想定読者や目指す状態などを設定しました。 今回から AWS の機械学習サービスの具体的な内容を見ていきたいと思います。

なお、2019年12月1日〜6日にアメリカのラスベガスにて AWS の最大のイベントである AWS re:Invent 2019 が開催されました。 ここで多数の新サービスや既存機能のアップデートが発表されましたので、それを踏まえて整理していきます。

今回扱う内容と目標¶

  • 「AWS の機械学習サービスの種類、それぞれのサービスの概要や使い方」がわかり、他者に簡単に説明できる状態になる。

AWS の機械学習サービスとは¶

AWS の機械学習サービスについて順を追って整理していきます。

機械学習分野における AWS のミッション¶

AWS の機械学習サービスについて正しく知るためには、まず機械学習分野における AWS のミッションを知る必要があります。 それは、機械学習サービスの製品紹介ページにて宣言されている通り「すべての開発者に機械学習の恩恵を」です。 それを踏まえて、機械学習サービスの種類を見ていきます。

(出典：機械学習サービスの製品紹介ページ)

機械学習サービスの種類¶

機械学習サービスには大きく分けて下記の表に示す3種類に分けることができます。 それぞれの機械学習サービスの特徴と利用用途を下記の表にまとめます。

機械学習サービスの選択の仕方¶

  • 仮説：種類がたくさんあってどれを使って良いのかわからないのでは？

  • それぞれのサービスの内訳を見る前に選択基準を整理する。

    • フローチャートか何かで選択基準を提案する

    • (※ 2回目はここで切れそう)

  • それぞれの特徴

    • 3つのサービスの内訳を見ていく

    • それぞれのサービスを一言で定義する

    • 詳細はサービスの紹介ページや開発者ガイドを見てもらう

      • サービスの紹介ページと Black Belt はリンクを紹介する

Amazon ECS
==================

基本事項
-------------

Task Definition
^^^^^^^^^^^^^^^^^^^^^
- Task を構成するコンテナ群の定義のこと。

    - コンテナ定義
    - CPU/MEM
    - Task に割り当てる IAM ロール
    - ネットワークモード
    - etc.

Task
^^^^^^^^^^^^
- Task Definition に基づいて起動されるコンテナ群。
- Task 内コンテナは同一ホスト上で実行される。
